# 🎵 Gesture-Based Music App

A Flutter-based mobile application that allows users to control music playback using intuitive hand gestures. The app aims to provide a hands-free and accessible music experience by replacing traditional touch controls with gesture recognition.

## Features

* 🎶 Play and pause music using gestures
* ⏭️ Skip to the next track
* ⏮️ Return to the previous track
* 🔊 Adjust volume with hand gestures
* 📱 Clean and responsive Flutter user interface
* ⚡ Fast and intuitive gesture-based interactions

## Tech Stack

* **Framework:** Flutter
* **Language:** Dart
* **Platform:** Android (with potential support for iOS)
* **Gesture Recognition:** Device camera and gesture detection libraries/APIs
* **Audio Playback:** Flutter audio packages

## Getting Started

### Prerequisites

* Flutter SDK
* Dart SDK
* Android Studio or Visual Studio Code
* Android Emulator or a physical Android device

### Installation

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/MobileAppDev.git
```

2. Navigate to the project directory:

```bash
cd MobileAppDev
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the application:

```bash
flutter run
```

## Usage

1. Launch the application.
2. Grant the required permissions (such as camera access if gesture recognition uses the device camera).
3. Select or start playing music.
4. Use the supported hand gestures to control playback.
